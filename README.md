# Krumbit Site

Official product website for [Krumbit: Learn While Scrolling](https://apps.apple.com/us/app/krumbit-learn-while-scrolling/id6747910126).

Krumbit turns the familiar rhythm of short-form video into a microlearning loop: watch a bite-sized lesson, take a quick quiz, choose the topics you care about, and build visible progress through streaks, XP, and learning stats.

## Development

Requires Node.js 22.13 or newer.

```bash
npm install
npm run dev
npm run build
npm test
```

The site includes the main product page plus `/privacy` and `/support` routes.
